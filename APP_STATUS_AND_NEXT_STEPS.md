## Next Steps

### To Run the App

You need Android device/emulator:

```bash
# Option 1: Android Emulator
flutter emulators launch emulator_name
cd "c:\Quiz App"
flutter run

# Option 2: Physical Phone (USB connected)
flutter run

# Option 3: Preview on Windows Desktop
flutter run -d windows
```

### To Deploy Live

**On Google Play Store**:

```bash
# 1. Build
flutter build appbundle --release

# 2. Create Play Store account ($25)

# 3. Upload to Play Console
# (Use web interface at play.google.com/console)

# 4. Submit for review

# 5. Goes live in 24-48 hours
```

**On Web Domain**:

```bash
# 1. Build APK
flutter build apk --release

# 2. Create simple HTML download page

# 3. Host on Netlify, GitHub Pages, or your domain
```
